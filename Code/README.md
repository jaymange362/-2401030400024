
  # Sustainable Travel Website Prototype

  This project is now a plain HTML, CSS, and JavaScript website (no React app runtime).

  ## Project structure

  - `index.html` contains the site shell (header, content region, footer).
  - `style.css` contains all page styles.
  - `script.js` handles navigation and renders pages in vanilla JavaScript.

  ## Run locally

  1. Install dependencies:

    npm install

  2. Start development server:

    npm run dev

  3. Optional production preview:

    npm run build
    npm run preview
